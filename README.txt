This application can encrypt and decrypt files through CMD commands.

Use command Vault encrypt <filepath> to encrypt file --> Get encryption key as output

Use command Vault decrypt <filepath> to decrypt file --> will be prompted for encryption key

Use command Vault secure <filepath> to have file encrypted and key sent to your email --> will be prompted for email address
